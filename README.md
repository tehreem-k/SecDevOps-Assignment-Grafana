



# Grafana Practice 

This repository contains proof of work for getting started with grafana

---

## Prerequisites
- Linux VM or EC2 instance
- Git installed on host
- Docker & Docker compose installed on host  

---

## Clone the repository

```bash
git clone https://github.com/grafana/tutorial-environment.git
cd tutorial-environment

```

## Run the Container
 
```bash
docker-compose up -d

```

**2. Browse and Try**

http://localhost:8081
